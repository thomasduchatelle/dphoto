Design Direction Summary

Visual Language: Modern minimal with sci-fi/gaming UI influences. Thin contrasting lines (1-2px) create structure without heavy chrome. Lines can have gaps or incomplete edges for visual interest.

Color Strategy: Dark theme (#121212) makes photos the primary focus. Brand blue (#185986) for all interactive elements, focus states, and primary actions. Thin accent lines use desaturated cyan/blue (#4a9ece).

Typography: Roboto with light/regular weights (300-400). Generous spacing for readability on dark backgrounds. Monospace for dates/metadata adds futuristic feel. Uppercase section labels with letter-spacing.

Spacing & Layout: Generous breathing room - photos need space. 8px base unit with 16-24px component spacing. Asymmetric layouts acceptable. Content floats against dark background rather than rigid grid confinement.

Key Differentiators: Incomplete line borders (don't span full width), gaming-style centered messages with just top/bottom lines, minimal chrome everywhere, subtle blue glows on hover/focus, no rounded corners on anything.

Prototyping Note: This aesthetic is distinctive but unproven. Start with one component (album card or error dialog) to validate it works. If thin lines feel too sparse or hard to read, dial back to more standard Material UI with subtle line accents while keeping dark theme and brand color.
